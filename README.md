# PhobOS Releases

This is a repository for hosting the release assets of [PhobOS](https://github.com/gaiaBuildSystem/cookbook-phobos).

If you want the OS images they are stored at: https://github.com/gaiaBuildSystem/phobos-releases

This repo stores the:

- vmlinux;
- vmlinux with debug symbols;
- .config file used to build the kernel;
- .config file used to build u-boot;
- SBOM (Software Bill of Materials) for the release;
